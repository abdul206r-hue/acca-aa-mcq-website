# MCQ Formatting Guide

This guide provides comprehensive instructions on how to format all five types of MCQ questions: STANDARD MCQ, MATCHING, MULTI_SELECT_ROWS, MULTI_SELECT_COLUMNS, and SELECT_TWO.

## 1. STANDARD MCQ
   **Format:**  A question followed by four options, with one correct answer.
   
   **Example:**  
   **Question:** What is the capital of France?  
   A) Berlin  
   B) Madrid  
   C) Paris  
   D) Rome  
   **Correct Answer:** C)

## 2. MATCHING
   **Format:** A list of items that need to be matched with another list.
   
   **Example:**  
   **Match the country with its capital:**  
   1. Germany  
   2. Italy  
   3. Spain  
   4. France  
   
   A) Rome  
   B) Paris  
   C) Berlin  
   D) Madrid  
   
   **Correct Matches:**  
   1-C, 2-A, 3-D, 4-B

## 3. MULTI_SELECT_ROWS
   **Format:** A question with multiple rows where each row can have one or more correct answers.
   
   **Example:**  
   **Select all that apply:**  
   | Item        | Option 1 | Option 2 |
   |-------------|----------|----------|
   | Fruit      | Yes      | No       |
   | Vegetable  | Yes      | No       |
   | Grain      | Yes      | No       |
   
   **Correct Answers:** Option 1 for Fruit and Vegetable.

## 4. MULTI_SELECT_COLUMNS
   **Format:** A matrix where multiple choices can be selected for each option.
   
   **Example:**  
   **Select all that apply:**  
   |            | Yes | No  |
   |------------|-----|-----|
   | Option 1   |     |     |
   | Option 2   |     |     |
   | Option 3   |     |     |
   
   **Correct Answers:** Yes for Option 1 and Option 3.

## 5. SELECT_TWO
   **Format:** A question where two correct answers must be selected from a list.
   
   **Example:**  
   **Select two correct answers:**  
   **Question:** Which of the following are programming languages?  
   A) Python  
   B) HTML  
   C) Java  
   D) CSS  
   **Correct Answers:** A) Python and C) Java.

This guide should help you format your 200 MCQs accurately.
